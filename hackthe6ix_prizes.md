# Hack the 6ix 2026 Prize Categories & Requirements

Here are all the prize categories extracted from the Hack the 6ix 2026 Devpost, along with their specific requirements and guidelines constraints.

## Main Prizes
*   **Hack the 6ix: 1st Place** *(Gotrax Edge Hoverboard, MLH Winner Pin)*
    *   **Requirement:** Best overall project created at Hack the 6ix 2026.
*   **Hack the 6ix: 2nd Place** *(Google Fitbit Air Smart Watch, MLH Winner Pin)*
    *   **Requirement:** 2nd best overall project at Hack the 6ix 2026.
*   **Hack the 6ix: 3rd Place** *(Portable Standalone Projector, MLH Winner Pin)*
    *   **Requirement:** 3rd best overall project at Hack the 6ix 2026.

## General Categories
*   **Hack the 6ix: Best Hardware Hack** *(High-quality Studio Speakers)*
    *   **Requirement:** Best hardware-centered project.
*   **Hack the 6ix: Best Environmental Hack** *(IKEA BLAHAJ Stuffed Animal)*
    *   **Requirement:** Best environmental, sustainability-focused project.
*   **Hack the 6ix: Best Beginner Hack** *(Mechanical Gaming Keyboard)*
    *   **Requirement:** More than 50% of team members must be attending Hack the 6ix 2026 as their first hackathon.
*   **Hack the 6ix: Best Predict the 6ix Trading Bot** *(Prize)*
    *   **Requirement:** Build a bot to interact with prediction markets on the dashboard (max 5 bots per user). The user whose bot has the most money at the end of the event wins.

## Sponsored Prizes
*   **Base44: Base44 Venture Builder Challenge** *($2,000 / $1,000 CAD)*
    *   **Requirement:** Use Base44 to build and launch an AI-powered product, business, or workflow that demonstrates real customer value.
*   **QNX: Best Use of QNX** *($1,000 / $250 CAD)*
    *   **Requirement:** Create an AI-powered embedded system with the QNX real-time operating system.
*   **Chexy: Make Every Payment Count with Chexy** *(Private Review, Priority Resume)*
    *   **Requirement:** Build a working prototype demonstrating at least one end-to-end payment-related workflow using sandbox/synthetic data (NO real money/credentials). Must clearly identify the target user, explain security/privacy, and define the financial benefit. Direct API integration with Chexy is not required.
*   **Deloitte: Green AI, and AI for Green** *(Meta Quest / Meta Ray-Ban AI glasses)*
    *   **Requirement:** Design AI-powered solutions to advance sustainability. Focus areas: Green AI (energy-efficient systems) and AI for Green (emissions reduction, climate prediction, resource optimization).
*   **Warp: Best Developer Tool** *(Keychron mechanical keyboards)*
    *   **Requirement:** Build a hack that improves the developer experience (creating, modifying or testing software). You do not need to use Warp. 
*   **Freesolo: Best Model Trained on Freesolo** *(Flight to SF for interview)*
    *   **Requirement:** Train an LLM using Freesolo’s post-training platform (SFT, RL, distillation techniques) for a single or multi-turn model with an interesting use case.
*   **CORTEX BioSciences: GROUD TRUTH** *($1,000 & work trial)*
    *   **Requirement:** Build a system that updates beliefs correctly from a given knowledge base when stream of new experimental results arrive (revising when real, holding firm when hype, isolating bad actors/instructions).
*   **Qualcomm: Build at the Edge with Arduino UNO Q** *(Meta Quest / Meta Ray-Ban AI glasses)*
    *   **Requirement:** Build a cloud-free edge solution showing an intentional split between intelligence (local AI/ML on Linux processor side) and reaction (real-time sensing/control on MCU side).
*   **Stay22: Peak Unhinged Big Brain Use of the Stay22 API** *($800 / $50 Visa gift cards)*
    *   **Requirement:** Use the Stay22 live accommodation API creatively—do not build another booking carousel. "Take hotel data somewhere it has no business being".
*   **Backboard: Most Innovative use of Backboard.io** *($200 CAD + ~$1000 credits)*
    *   **Requirement:** Use Backboard to route across AI models, handle embeddings, RAG, tool calls, and orchestration for a compelling new tool or feature.
*   **Pheobe: AI to Coordinate the Real World** *($100 CAD)*
    *   **Requirement:** Build an AI teammate/agent that automates workflows, improves productivity, or helps people make better decisions in the real world.
*   **Blockchain for Good (BGA): AI Trading Strategies for Financial Inclusion** *($400 USD)*
    *   **Requirement:** Build an AI-assisted trading or investing strategy addressing a specific financial inclusion problem (e.g. exploitation of inexperienced investors, barriers to entry, opacity in AI-advice). Must name specific harm and who it affects.
*   **Unifold: Best Use of Unifold** *($1,000 USD)*
    *   **Requirement:** Use the Unifold SDK to power stablecoin deposits or payments in your app with a creative, functional integration.
*   **ElevenLabs: Best Project Built with ElevenLabs** *(6-month Scale tier)*
    *   **Requirement:** Creative use of ElevenLabs API for autonomous agents beyond simple TTS. Evaluated on interaction design, multimodal implementations, and prompt engineering.

## MLH Partner Prizes
*   **MLH: Best Use of MongoDB Atlas** *(M5GO IoT Starter Kit)*
    *   **Requirement:** Build a hack using MongoDB Atlas as a cloud database.
*   **MLH: Best Use of Auth0** *(Wireless Headphones)*
    *   **Requirement:** Enable user authentication (sign-in, MFA, etc.) into your application using any of the Auth0 APIs.
*   **MLH: Best Use of Gemini API** *(Google Swag)*
    *   **Requirement:** Build AI-powered apps with the Google Gemini API (language models, chatbots, summarization, creative content).
*   **MLH: Best Use of Solana** *(Ledger Nano S Plus)*
    *   **Requirement:** Innovate with the Solana blockchain, harnessing fast execution and near-zero transaction costs.
*   **MLH: Best Use of Presage** *(Fitbit Inspire & Presage Perks)*
    *   **Requirement:** Integrate the Presage SDK to leverage human sensing (real-time heart rate, breathing rate, engagement, facial expressions) through standard cameras.
